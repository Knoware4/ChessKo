## Pre-requisites

- Node.js (recommended LTS, e.g. 18+)
- npm (part of Node.js)
- Firebase project with service account for Firebase Admin SDK

---

- You need to set your Firebase configuration in the *frontend/src/firebase.ts* file
- How to get the Firebase configuration for your project's application: https://support.google.com/firebase/answer/7015592?hl=en#zippy=%2Cin-this-article
- For getting the Firebase configuration you need to set up the firebase project with the service account
- You can find the instructions for setup of the Firebase project with the service account *in the repository's root directory README.md file*

### 1. Install dependencies

```bash
npm install
```


### 2.1 Run the frontend

```bash
npm start
```

### 2.2 Optional build
```bash
npm build
```
